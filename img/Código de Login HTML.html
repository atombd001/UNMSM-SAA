Para referenciar un archivo JavaScript que se encuentra en una carpeta diferente a la del archivo HTML, debes usar la ruta relativa o absoluta del archivo en el atributo src de la etiqueta <script>.

Ejemplo de rutas:

  1. Si el archivo JS está en una carpeta llamada js dentro del mismo directorio que tu archivo HTML:
html
<script src="js/login.js"></script>
  1. Si el archivo JS está en una carpeta superior (por ejemplo, el archivo HTML está en carpeta1 y el archivo JS está fuera de esa carpeta):
html
<script src="../login.js"></script>

En este caso, .. significa que vas a un nivel superior en el árbol de directorios.

  1. Si el archivo JS está en una carpeta completamente diferente (por ejemplo, scripts en la raíz del proyecto):
html
<script src="/scripts/login.js"></script>
  1. Si estás usando una ruta absoluta con un dominio o servidor externo:
html
<script src="https://example.com/js/login.js"></script>

Ejemplo más completo:

Supongamos que tienes la siguiente estructura de carpetas:

markdown
proyecto/
│
├── css/
│   └── login.css
├── js/
│   └── login.js
└── paginas/
    └── login.html

El archivo HTML (login.html) está en la carpeta paginas/, mientras que el archivo JS está en la carpeta js/. En este caso, dentro de tu archivo login.html, la referencia al archivo login.js sería:

html
<script src="../js/login.js"></script>

Este ejemplo usa .. para salir de la carpeta paginas y luego entrar en la carpeta js para acceder al archivo login.js.

  // Validar email y contraseña
  if (!validateEmail(email)) {
    alert('Por favor, introduce un correo electrónico válido.');
    return;
  }

  if (!validatePassword(password)) {
    alert('La contraseña debe tener al menos 8 caracteres.');
    return;
  }

  // Simulación de envío de datos al servidor (aquí podrías usar AJAX o fetch)

  // Redirigir a la página principal

function validateEmail(email) {
  // Expresión regular para validar el formato del email
  const re = /^[^\s@]+@[^\s@]+\.[^\s@]+$/;
  return re.test(email);
}

function validatePassword(password) {
  // La contraseña debe tener al menos 8 caracteres
  return password.length >= 8;
}

  • Redirección: Después de la validación, redirige al usuario a la página de inicio (en este caso, /).
